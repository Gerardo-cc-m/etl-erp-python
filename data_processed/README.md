# Processed Data

This folder contains the processed and consolidated datasets generated by the ETL pipeline.

Characteristics of the processed data:
- Monthly aggregated sales data
- Harmonized product, importer, and country dimensions
- Business rules applied (brand logic, currency conversion, filtering)
- Data ready for reporting and analytics tools (Power BI, Excel)

Note:
All datasets included here are fictitious and were created exclusively for demonstration and portfolio purposes.
