# Raw Data

This folder represents the raw input data extracted from ERP systems (e.g. SAP).

In a real production environment, this directory contains:
- Monthly sales/order extracts downloaded from SAP
- Text or CSV files with custom delimiters and headers
- Files with no prior transformation or cleansing applied

⚠️ Note:
Real raw data files are not included in this repository due to confidentiality constraints.
The structure and processing logic can be reviewed in the ETL scripts.
